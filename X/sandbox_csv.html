Imapsync CSV online

This Imapsync CSV online service is a prototype, a working prototype.

Some explanations about this service:
  • The input textarea is multi-lines, each line describe a synchronization from the source account to the destination account.
  • Blank lines are ignored. Blanks lines have only spaces or tabs or nothing but a CR
  • Lines beginning with # are also ignored, use that for commenting.
  • A basic line will sync all messages in all folders from the source account to the destination account.
  • Each line has seven columns, six for the credentials, and one extra to add one or several parameters to the imapsync command.
  • Each line has seven semi-colons characters ";", each one delimiting a column. The last column can be empty.
  • Each line is processed serially, one synchronization at a time.
  • At the end of the log, the lines that failed are listed.
  • The input textarea is prefilled with values, as an example, replace them with yours or remove them.
